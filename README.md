# 花札 こいこい (Hanafuda Koi-Koi)

一个基于Web的日本花札（花札/はなふだ）游戏，支持标准Koi-Koi规则的双人在线对战。完全中文界面。

## 游戏简介

花札是日本传统的纸牌游戏，由48张牌组成，分为12个月份，每月4张牌。Koi-Koi是花札最流行的玩法之一，两人对战，通过配对相同月份的牌来获得牌组，并通过组成特定的"役"（牌型组合）来得分。

## 功能特性

- 🎴 标准48张花札牌（完全中文命名）
- 🎯 完整的Koi-Koi规则实现
- 🏆 支持所有主要役种（五光、四光、三光、猪鹿蝶、赤短、青短等）
- 🌐 实时双人在线对战
- 💻 响应式Web界面（完全中文）
- 🎨 精美的视觉设计
- ✨ **每张牌都有类型标识**（光、种、短、滓）
- 📊 **获得牌按类型自动分类显示**
- 🎲 **完整的Koi-Koi倍数系统**（每次喊Koi-Koi倍数+1）
- 🎯 **实时显示已达成的役种和当前倍数**
- ✅ 单个玩家断线时，房间保留5分钟等待重连

## 牌的类型

游戏中的48张牌分为4种类型：

| 类型 | 图标 | 说明 | 数量 |
|------|------|------|------|
| 光 | ✨ | 光札，最高级的牌 | 5张 |
| 种 | 🦋 | 种札，带有动物或物品 | 9张 |
| 短 | 🎀 | 短册札，带有诗句的彩带 | 10张 |
| 滓 | 🍃 | 滓札，普通牌 | 24张 |

## Koi-Koi倍数系统

当你达成役种时，可以选择：
- **こいこい（Koi-Koi）**: 继续游戏，你的倍数+1，下次达成役时得分会翻倍！
- **胜利**: 立即结束回合，获得当前的分数（基础分×倍数）

**风险**: 选择Koi-Koi后，如果对手先达成役，你将失去本回合所有分数！

## 安装和运行

### 前置要求

- Node.js 14.0 或更高版本
- npm 或 yarn

### 安装步骤

1. 克隆或下载项目到本地

2. 安装依赖：
```bash
npm install
```

### 启动服务器

```bash
npm start
```

服务器将在 `http://localhost:3000` 启动。

### 开始游戏

1. 在浏览器中打开 `http://localhost:3000`
2. 打开另一个浏览器窗口（或让朋友访问你的IP地址）
3. 两个玩家都会自动匹配到同一个游戏房间
4. 游戏自动开始！

## 游戏规则

### 游戏流程

**每个回合分为两个阶段：**

1. **出手牌阶段**:
   - 从手牌中选择一张牌打出
   - 如果场上有相同月份的牌，需要配对收取
   - 如果有多张相同月份的牌，需要选择一张配对

2. **抽牌阶段**:
   - 从牌堆抽一张牌
   - 同样需要与场上相同月份的牌配对
   - 完成后检查是否达成役种


3. **Koi-Koi决策**:
   - 当你获得的牌组成了一个或多个"役"时，游戏会暂停
   - 你可以看到已达成的役种和当前倍数
   - 选择：
     - **こいこい（Koi-Koi）**: 继续游戏，倍数+1，争取更高分数
     - **胜利**: 结束回合，获得 基础分×倍数 的分数
   - 选择Koi-Koi后，如果对手先达成役，你将失去所有分数！

4. **回合结束**:
   - 当有玩家选择结束游戏时
   - 或牌堆用尽时
   - 显示双方的役种、倍数和得分
   - 计算总分

## 界面功能

### 游戏界面说明

1. **玩家信息区**:
   - 显示当前得分和倍数
   - 显示已达成的役种（实时更新）
   - 按类型分类显示获得的牌：光、种、短、滓

2. **场地区域**:
   - 显示场上的8张牌
   - 显示剩余牌堆数量
   - 显示当前回合状态

3. **手牌区域**:
   - 显示你的手牌
   - 轮到你时可以点击出牌
   - 每张牌都有类型标识（光/种/短/滓）

4. **对手信息区**:
   - 显示对手得分、倍数和手牌数
   - 显示对手已达成的役种
   - 按类型显示对手获得的牌

### 牌的显示

每张牌包含以下信息：
- 类型图标（✨光/🦋种/🎀短/🍃滓）
- 类型标识（光/种/短/滓）
- 月份
- 牌名

### 游戏技巧

- 记住已经打出的牌
- 注意对手收集的牌类型，判断对手可能的役种
- 在合适的时机选择Koi-Koi
- 计算风险和收益：倍数越高，风险越大
- 注意保护自己的役种组合，避免对手反超
